Milligram demo

The Grid is a fluid system with a max width of 112.0rem (1120px), that shrinks with the browser/device at smaller sizes. The max width can be changed with one line of CSS and all columns will resize accordingly.

The base type is 1.6rem (16px) over 1.6 line height (24px)
